# Labs

Labs are small experiments for validating collection, verification, scoring,
and Browser Use workflows.

Suggested first labs:

1. Compare one source through API/RSS and browser observation.
2. Use a lead source to find a candidate item, then verify it from the original
   source.
3. Generate a five-item daily radar from a fixed source list.
4. Record failure modes from dynamic pages.
5. Convert one weekly theme into a practice task.
